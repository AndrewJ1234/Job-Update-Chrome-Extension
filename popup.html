Setup Instructions:

  1. Go to notion.so/my-integrations
  2. Create a new integration and copy the token
  3. Share your database with the integration
  4. Copy the data source ID from your database URL
  5. Ensure your database has these properties:
    • Position (Title)
    • Company (Text)
    • Industry (Text)
    • Application Status (Select)
    • Applied (Date)
    • Interview Date (Date)
    • Application Link (URL)
    • Note (Text)
